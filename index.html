CSS review notes

盒子模型部分补充知识点
  1. width：-webkit-fill-available 自动撑满空间；
  2. width: fit-content 根据内容自适应宽度；
  3. width: max-conten 根据内容自适应（按照最大的子元素内容长度）
背景处理部分补充知识点

  1. background-clip:

    • content-box; 背景只包括内容区域
    • padding-box; 背景包括内容和内边距
    • border-box; 背景延伸到边框

  2. background-repeat:

    • repeat: 设置背景图片（background-img）时，重复
    • space: 平均分配
    • no-repeat: 不重复

  3. background-attachment:

    • fix 有滚动条时不随滚动条滚动
    • scroll 随滚动条滚动

  4. background-position:

    • center 居中
    • bottom right 右下
    • 50% 40% 距离上边50%, 距离右边40%
    • 100px 20px 距离上边100px, 距离右边20px

  5. background-size:

    • 100% 100% 设置背景的宽度，高度的具体值或比例
    • cover 保证背景图片覆盖满盒子（撑满）
    • contain 保证背景图片完全显示（可能有留白）

  6. background-img: url(),url(),… 背景图片多个

  7. background: red url() no-repeat center （一行语句）

  8. 盒子阴影

    • box-shadow: 10px 10px 10px rgba(100,100,100,.2);
      各个属性值分别为水平偏移量，垂直偏移量，柔和度，颜色

  9. 背景渐变色：

    • 线性渐变
      background: liner-gradient(90deg, red, green, blue)
    • 镜像渐变
      bacakground: radial-gradient(at 50% 50%, red, green, blue)
    • 渐变标志位(绘制小太阳)
      bacakground: radial-gradient(red, yellow 30%, black 70%, black 100%)
数据元素样式补充知识
  1. 表格标题标签：table title
  2. 单元格间距：border-spacing
  3. 表格细线：border-collapse:collapse
  4. 有序列表ul列表编号风格属性：
    -list-style-tyle：
    可以设置为none,取消默认风格，该属性会继承
    -list-style-imag
    也可以在li标签中用背景图片，调整位置作为列表编号
  5. 伪元素before,after样式：当作独立元素对待
浮动布局补充知识
  1. 浮动元素会脱离文档流
  2. ==行级元素浮动后转为块元素,可以设置宽高==
  3. 清除浮动：
    • clear:left 清除该元素左侧浮动元素的影响
    • clear:right 清除该元素左侧浮动元素的影响
    • clear:both 清除该元素两侧侧浮动元素的影响
  4. 清除浮动的方法：
    • 补充一个空元素
    • 使用伪元素after
    • 利用触发BFC机制清除浮动,
      ==触发BFC后，父元素计算高度时，浮动子元素高度也参与计算==
  5. 环绕距离控制：
    • shape-outside: border-box;
    • shape-outside: margin-box;
    • shape-outside: content-box;
    • shape-outside: padding-box;
  6. 浮动显示区域形状控制：
    • clip-path: circle(50% at 0 0)
    • clip-path: ellipse(20% 30%)
    • clip-path: polygon(50% 0, 50% 100%, 0 50%)
  7. 内容环绕排版：shape-outside 和 clip-path 属性值一致
定位补充知识
  1. 相对定位：保留空间位，相对自己在文档中的原位置
  2. 绝对定位：不保留空间位，相对最近的开启了定位的父元素，子绝父相
  3. 当元素没有宽高时，通过定位可以设置宽高
  4. ==通过定位实现元素居中的几种方式==
    • left:0.5*（父盒子宽度-子元素宽度), 垂直居中同理
    • left: 50% ; margin-left:-0.5*子元素宽度，垂直居中同理

5.定位叠加的层级：z-index，属性值越大优先级越高

6.固定定位：相对于浏览器窗口进行定位，脱离原来的文档流
- 使用固定定位做顶部导航栏时，下面的主体内容会穿透导航 栏，主体内容盒子需要设置maigin-top；

  1. 粘性定位：sticky,
    • 可以用来把文章标题固定在顶部，不随文 章内容滚动
    flex布局补充知识
弹性盒子
  1. 把普通盒子变成弹性盒子
    • display:flex;
  2. 盒子中弹性元素的方向
    • flex-direction:row (默认从左到右)
    • flex-direction:row-reverse
    • flex-direction:column
    • flex-direction:column-reverse
  3. 盒子中的元素超过一行时排列
    • flex-wrap:nowrap; ==（默认情况下会把弹性元素变小排在一行）==
    • flex-wrap: wrap 从下往上折行
    • flex-wrap: wrap-reverse; 从下往上折行
    • 如果想从上往下或从下往上，改变flex-direction的方向
  4. 弹性元素延主轴的排列方式
    • justify-content: flex-start; (默认)
    • justify-content: flex-end;
    • justify-content: center
    • justify-content: space-between;(左右靠边)
    • justify-content: space-around;(左右两侧都有间距)
    • justify-content: space-evenly;(完全平均分布)
  5. 弹性元素延交叉轴的排列方式
    • aline-items: flex-start; （默认）
    • aline-items: flex-end;
    • aline-items: center;
    • aline-items: stretch; ==(拉伸对其交叉轴两端，优先级低于元素设置的尺寸)==
  6. ==多行元素延交叉轴的排列方式==
    • 默认情况下两行有间距
    • align-content：flex-start;
    • align-content: flex-end;
    • align-content: center;
    • align-content: space-between;
    • align-content: space-around;
    • align-content: space-evenly;

hexo + gitup 搭建个人博客

hexo + gitup 搭建个人博客

安装

  • 环境准备：node， git
  • 利用npm安装hexo：cnpm install -g hexo-cli

hexo常用命令

  • 初始化博客： hexo init
  • 本地启动： hexo s
  • 新建一篇博客：hexo n “my first blog”
  • 清除缓存：hexo clean
  • 生成博客: hexo g
  • 推向github: hexo d
