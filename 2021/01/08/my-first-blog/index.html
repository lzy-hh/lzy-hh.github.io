hexo + gitup 搭建个人博客

安装

  • 环境准备：node， git
  • 利用npm安装hexo：cnpm install -g hexo-cli

hexo常用命令

  • 初始化博客： hexo init
  • 本地启动： hexo s
  • 新建一篇博客：hexo n “my first blog”
  • 清除缓存：hexo clean
  • 生成博客: hexo g
