CSS review notes

CSS补充知识
盒子模型部分补充知识点
  1. width：-webkit-fill-available 自动撑满空间；
  2. width: fit-content 根据内容自适应宽度；
  3. width: max-conten 根据内容自适应（按照最大的子元素内容长度）
背景处理部分补充知识点

  1. background-clip:

    • content-box; 背景只包括内容区域
    • padding-box; 背景包括内容和内边距
    • border-box; 背景延伸到边框

  2. background-repeat:

    • repeat: 设置背景图片（background-img）时，重复
    • space: 平均分配
    • no-repeat: 不重复

  3. background-attachment:

    • fix 有滚动条时不随滚动条滚动
    • scroll 随滚动条滚动

  4. background-position:

    • center 居中
    • bottom right 右下
    • 50% 40% 距离上边50%, 距离右边40%
    • 100px 20px 距离上边100px, 距离右边20px

  5. background-size:

    • 100% 100% 设置背景的宽度，高度的具体值或比例
    • cover 保证背景图片覆盖满盒子（撑满）
    • contain 保证背景图片完全显示（可能有留白）

  6. background-img: url(),url(),… 背景图片多个

  7. background: red url() no-repeat center （一行语句）

  8. 盒子阴影

    • box-shadow: 10px 10px 10px rgba(100,100,100,.2);
      各个属性值分别为水平偏移量，垂直偏移量，柔和度，颜色

  9. 背景渐变色：

    • 线性渐变
      background: liner-gradient(90deg, red, green, blue)
    • 镜像渐变
      bacakground: radial-gradient(at 50% 50%, red, green, blue)
    • 渐变标志位(绘制小太阳)
      bacakground: radial-gradient(red, yellow 30%, black 70%, black 100%)
数据元素样式补充知识
  1. 表格标题标签：table title
  2. 单元格间距：border-spacing
  3. 表格细线：border-collapse:collapse
  4. 有序列表ul列表编号风格属性：
    -list-style-tyle：
    可以设置为none,取消默认风格，该属性会继承
    -list-style-imag
    也可以在li标签中用背景图片，调整位置作为列表编号
  5. 伪元素before,after样式：当作独立元素对待
浮动布局补充知识
  1. 浮动元素会脱离文档流
  2. ==行级元素浮动后转为块元素,可以设置宽高==
  3. 清除浮动：
    • clear:left 清除该元素左侧浮动元素的影响
    • clear:right 清除该元素左侧浮动元素的影响
    • clear:both 清除该元素两侧侧浮动元素的影响
  4. 清除浮动的方法：
    • 补充一个空元素
    • 使用伪元素after
    • 利用触发BFC机制清除浮动,
      ==触发BFC后，父元素计算高度时，浮动子元素高度也参与计算==
